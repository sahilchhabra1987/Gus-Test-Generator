## PRD: Password Reset Flow

### Overview
Users who forget their password can reset it via email verification.

### Requirements
1. User clicks "Forgot Password" on login page
2. User enters their registered email address
3. System sends reset email within 60 seconds
4. Reset link is valid for 24 hours
5. User sets new password (min 8 chars, at least 1 number, 1 uppercase)
6. Old password is invalidated immediately after reset
